How does Local AI work?

Running a local AI means your computer is doing all the thinking. No data is sent to the cloud, it's 100% private, and completely free! We use an incredible tool called Ollama that lets you run large language models on your own machine.

Setup Instructions (Step-by-Step):

  1. Download Ollama: Install Ollama for Windows.
  2. Open Command Prompt: Press Win+R, type cmd, and press Enter.
  3. Download the Brain: Paste this command and hit Enter to download qwythos-9b (this may take a few minutes depending on your internet speed):
    ollama run qwythos-9b
  4. Fix Windows Blocking (CORS): If Luna says 'Connection Failed', Windows might be blocking it. Open Command Prompt as Administrator and run:
    setx OLLAMA_ORIGINS "*"
    Then restart your PC or completely close and reopen the Ollama app from the system tray.
  5. Configure Luna: Inside Luna, go to Settings (Gear Icon) -> Luna AI -> Enable "Use Local Model" and type qwythos-9b in the Model Name box.
